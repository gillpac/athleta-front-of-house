# Athleta Front of House

Custom lead, trial, sales & retention system for Athleta Gymnastics (Coolaroo + Altona North).

- `CLAUDE.md` — standing brief for every Claude Code session
- `docs/SCOPE.html` — the v1 build scope (source of truth)
- `docs/PROTOTYPE_REFERENCE.jsx` — frozen UI reference (v10)
- `supabase/schema.sql` — database schema
- `tasks/` — numbered task briefs; one per Claude Code session, in order

## How to run a build session
1. Go to claude.ai/code, pick this repo
2. Paste: "Do tasks/0X-name.md"
3. Review the PR via its Vercel preview link; merge when happy
